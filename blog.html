Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerText will show the value as is and ignores any HTML formatting which may be included.InnerHTML will show the value and apply any HTML formatting.TextContent appears to the same as innerText but keeps formatting.

Question-3: What is event delegation in the DOM?

DOM event delegation is a mechanism of responding to ui-events via a single common parent rather than each child, through the magic of event "bubbling".

Question-4: What is event bubbling in the DOM?

Event bubbling in javascript is a mechanism where an event triggered on a child element propagates upward through its ancestors in the dom. It allows parent elements to respond to events triggered by their child elements..
